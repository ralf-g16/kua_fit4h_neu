#GUI gegen Non GUI (GUI - Graphical User Interface/Mode)
 
 -zwei verschiedene Möglichkeiten
 GUI:
 	-Fenster verschieben/ Größe ändern
 	-Anwendungen mit grafischen Übersichten, einfachere Strukturierung
 	-Multifenster Möglichkeiten
 	-gleiche Programme mehrfach öffnen
 	-Ressourcenintensiv

 Non-GUI:
 	-keine Fenster
 	-nur die Shell
 	-keine Multifenster Möglichkeiten
 	-alles Textbasiert
 	-spart Ressourcen

MOTD - Message of the Day - Mitteilung vom Admin für Benutzer

#Virtualization and Cloud Computing

-multiuser operating system
	-viele Benutzer können daran arbeiten ohne sich gegenseitig zu beeinflussen
	-unpriviligierte Benutzer

-Physikalischer Computer -> Host
 -stellen virtuelle Maschinen bereit -> Clients

 -Host Software wird Hypervisor genannt

 -Virtualisiert funktioniert, da die meisten Hosts im idle 'rumdümpeln'
 -Sie brauchen gute CPU's und viel RAM

 -gut da:
 	-Snapshots
 	-können leicht neu gemacht werden

 Cloud Computing

 -nur verbindung zum Host / VM
 -keinen physikalischen Zugriff
 -es werden nur für Ressourcen bezahlt
